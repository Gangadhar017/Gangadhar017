## ⚡ How to Setup Your New Profile README in 5 Minutes

### Step 1: Create Your Special GitHub Repository
1. Go to GitHub and click **New Repository**.
2. For the **Repository Name**, type your exact GitHub username: `Gangadhar017`.
   > 💡 *GitHub will show a special banner: "You found a secret! Gangadhar017/Gangadhar017 is a special repository that you can use to add a README.md to your GitHub profile."*
3. Make sure the repository is **Public**.
4. Check **Initialize this repository with a README**.
5. Click **Create Repository**.

### Step 2: Upload Your Premium Profile Banner
1. Open your newly created `Gangadhar017` repository on GitHub.
2. Click **Add file** -> **Upload files**.
3. Drag and drop the file `github_banner.png` (located in this scratch directory) into the browser.
4. Commit the changes directly to the `main` branch.
   > 💡 *Note: The README is configured to load this image directly via: `https://raw.githubusercontent.com/Gangadhar017/Gangadhar017/main/github_banner.png`. Once uploaded, it will load perfectly!*

### Step 3: Copy and Paste the Premium README Content
1. Open the `README.md` file generated in this directory (`gangadhar-github-profile/README.md`).
2. Copy the entire contents of the file.
3. On GitHub, open your `Gangadhar017/README.md` file and click the **pencil icon (edit)**.
4. Paste the copied contents, replacing any default text.
5. Click **Commit changes...** at the top right to save.
